# About

The **Citadel Wiki** is a static knowledge base.

- **Generator:** [MkDocs](https://www.mkdocs.org/) with the [Material](https://squidfunk.github.io/mkdocs-material/) theme
- **Source:** [github.com/ihavedna/citadel-wiki](https://github.com/ihavedna/citadel-wiki)
- **Hosting:** GitHub Pages, deployed via GitHub Actions

## Why not MediaWiki?

MediaWiki requires a PHP server and a SQL database, which GitHub Pages (a static host) cannot provide. MkDocs Material gives a wiki-like experience — navigation, search, and browser-based editing — while building to plain static files that Pages can serve.
